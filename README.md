# znanie-drevnix-skills

Claude Code Skills для работы с языком программирования В†† (Ве крест крест).

## О языке В††

В†† — древнерусский язык программирования, построенный поверх C++ с помощью препроцессорных макросов. Все стандартные ключевые слова, типы данных и функции стандартной библиотеки заменены на древнерусские/славянские эквиваленты.

Исходный проект: [KanatnikovMax/znanie-drevnix](https://github.com/KanatnikovMax/znanie-drevnix)

## Доступные Skills

### `/cpp-to-vkk` — Преобразование C++ в В††

Преобразует код на C++ в эквивалентный код на языке В††.

**Использование в Claude Code:**
```
/cpp-to-vkk path/to/file.cpp
/cpp-to-vkk int main() { cout << "Hello"; return 0; }
```

**Что делает skill:**
- Заменяет все C++ ключевые слова на древнерусские эквиваленты
- Преобразует типы данных (`int` → `целина`, `void` → `бестолочь`, `string` → `много_букав` и т.д.)
- Переводит функции стандартной библиотеки (`cout` → `молвить`, `cin` → `внемлить` и т.д.)
- Переименовывает переменные и функции на русский язык в древнерусской стилистике
- Переводит комментарии на русский язык
- Добавляет корректный заголовочный файл `Ве_крест_крест.h`

## Примеры преобразования

### C++ (исходный код)
```cpp
#include <iostream>
using namespace std;

int main() {
    string message = "Hello World!";
    cout << message << endl;
    return 0;
}
```


## Основные замены (краткая справка)

| C++ | В†† | Описание |
|-----|-----|----------|
| `int main` | `царь_батюшка_главный` | Точка входа |
| `cout` | `молвить` | Вывод |
| `cin` | `внемлить` | Ввод |
| `if/else` | `коли/отнюдь` | Условия |
| `for` | `для` | Цикл |
| `while` | `покуда` | Цикл |
| `return` | `воздать` | Возврат |
| `class` | `династия` | Класс |
| `template` | `свет_мой_зеркальце` | Шаблон |
| `new/delete` | `новь/казнь` | Память |
| `true/false` | `правда/кривда` | Логические |
| `int` | `целина` | Тип |
| `void` | `бестолочь` | Тип |
| `string` | `много_букав` | Тип |

Полный список замен доступен в файле `.claude/commands/cpp-to-vkk.md`.
