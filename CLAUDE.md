# Знание Древних — Skills для Claude Code

Этот репозиторий содержит Claude Code Skills для работы с языком программирования В†† (Ве крест крест) — древнерусским языком программирования, построенным поверх C++ с помощью препроцессорных макросов.

## О языке В††

В†† — это обёртка над C++, заменяющая стандартные ключевые слова, типы и функции на древнерусские/славянские эквиваленты через `#define` и `using`. Код на В†† является валидным C++ кодом при подключении заголовочного файла `Ве_крест_крест.h`.

Исходный проект: https://github.com/KanatnikovMax/znanie-drevnix

## Доступные Skills

### `/cpp-to-vkk` — Преобразование C++ в В††

Преобразует код на C++ в эквивалентный код на языке В††. Заменяет все ключевые слова, типы, функции стандартной библиотеки и идентификаторы на древнерусские эквиваленты.

Использование:
```
/cpp-to-vkk <путь к файлу .cpp>
/cpp-to-vkk <код на C++>
```

## Структура репозитория

- `.claude/commands/` — Skill-файлы для Claude Code
  - `cpp-to-vkk.md` — Skill преобразования C++ → В††
- `header/` — Заголовочный файл языка В††
  - `Ве_крест_крест.h` — Основной заголовочный файл с макросами и типами
- `examples/` — Примеры кода на В††, демонстрирующие преобразование

## Сборка примеров

Примеры можно скомпилировать любым C++ компилятором с поддержкой C++11 и выше:

```bash
g++ -std=c++11 -o example examples/Слава_Руси.cpp
```

Заголовочный файл `Ве_крест_крест.h` должен находиться в директории, доступной компилятору (например, в `header/` с флагом `-I header/`):

```bash
g++ -std=c++11 -I header/ -o example examples/Слава_Руси.cpp
```
